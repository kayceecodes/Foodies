# Foodies - Restaurant Finder 
The App shows Restaurants of all sort in your area that based on you location & favorited selections.
It will also show suggestions. It allows users to sign in with Google api.

## API's
1. Mapbox API
2. Yelp API
3. Google Places API
4. Google Identity

## Tech
- .Net
- Next.js (React.js)
- Typescript
- Postgres
